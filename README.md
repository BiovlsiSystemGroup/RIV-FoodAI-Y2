# RIV-FoodAI-Y2
自行建立食品資料集，在RiscV開發版上實現模型推理與互動

## 教材設備:
HiFiv Unmatched Rev B x1 (搭配電源)
傳輸線 USB-A to TypeC  or USB-A to MicroUSB
MicroSD Card 32GB x1 (或以上)
microSD卡讀卡機
Webcam c270 x1
行動裝置 x1
電腦主機與螢幕x1
Wifi 路由器
(可選)RJ45 網路線
(可選)磅秤設備(CT1 + HX711)
(可選)M.2 WiFi 模組

## 教材內容：
第一章｜課程介紹與系統總覽
第二章｜拍攝資料集與食品成分表收集
第三章｜YOLO 模型訓練與辨識實作
第四章｜HiFiv Unmatched Rev B 開發板介紹與設定
第五章｜推理伺服器建置與系統整合
第六章｜推理伺服器與磅秤系統整合
